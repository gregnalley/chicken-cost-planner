# 🐔 Backyard Chicken Cost & Planning Tools

Free online calculators to help estimate the real cost of raising backyard chickens.

## Tools Included

- Chicken Feed Cost Calculator
- Egg Profit Calculator
- Chicken Coop Cost Estimator

## Start Here

👉 https://gregnalley.github.io/chicken-cost-planner/tools.html

## What This Site Does

This project helps backyard chicken owners estimate:
- feed costs
- coop startup costs
- egg production profit
- total flock expenses

All tools are free and run directly in the browser.
